# 아일랜드 백엔드 서버 개발

---

## 개발 기간

2022.02.14 ~ 2022.04.22 (총 3개월)

## 사용 기술

- Nest.js (Node.js Express)
- TypeORM
- MariaDB
- Linux (Ubuntu Server)
- Nginx
- Docker

## 프로젝트 기여도

100%

## 개발 부분

- 대규모 Bulk Insert 시, Bull.js와 Redis를 이용하여 성능 5배 이상 향상
- 로그 데이터 수집 및 개인 정보 암호화 처리 강화
- 메일 발송 처리 및 인증 체계 개선
