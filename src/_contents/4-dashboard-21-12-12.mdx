# 의약품 온도 관리 서버

---

## 개발 기간

2021.12 ~ 2021.12

## 사용 기술

- Nest.js (Node.js Express)
- TypeORM
- Nuxt.js (Vue.js)
- MariaDB
- Linux (Ubuntu Server)
- Nginx
- Docker

## 프로젝트 기여도

100% (풀스택)

## 담당 업무

- 태그 센서를 이용하여 의약품의 온도와 위치를 관리 및 추적할 수 있는 웹 백엔드 서버 및 대시보드 개발

## 개발 부분

- 서버 인프라 구축 (리눅스 우분투 서버)
- 백엔드 서버 개발 (Nest.js, Docker)
- 프론트엔드 개발 (Nuxt.js)
- 도커 게이트웨이를 이용하여 서버 로그를 웹훅 서버로 보내는 기능 구현 (ts-node로 개발)
- 서버 로그 제거 기능 추가
